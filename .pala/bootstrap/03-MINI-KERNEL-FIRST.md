# 03 — Mini Kernel First

Before big features, create the Pala kernel:

```txt
.pala/rules/
.pala/state/
.pala/ledger/
.pala/evidence/
docs/evidence/
```

Required first files:

```txt
.pala/rules/core-rules.md
.pala/rules/source-of-truth.yaml
.pala/rules/no-fake-done.md
.pala/rules/no-delete-no-push.md
.pala/rules/benchmark-policy.yaml
.pala/state/project-state.json
.pala/ledger/events.jsonl
.pala/ledger/handoffs.jsonl
```

This is repo memory. Agent memory is not enough.
