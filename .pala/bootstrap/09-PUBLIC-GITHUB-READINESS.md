Create public-ready files:

```txt
README.md
LICENSE
NOTICE
SECURITY.md
CONTRIBUTING.md
CODE_OF_CONDUCT.md
SUPPORT.md
GOVERNANCE.md
CHANGELOG.md
ROADMAP.md
docs/INSTALL.md
docs/MCP_CLIENTS.md
docs/ARCHITECTURE.md
.github/ISSUE_TEMPLATE/bug_report.yml
.github/ISSUE_TEMPLATE/feature_request.yml
.github/PULL_REQUEST_TEMPLATE.md
.github/workflows/ci.yml
.github/workflows/security.yml
.github/workflows/scorecard.yml
```

README rules:

- English-first
- honest status
- no fake npm/PyPI publish claim
- quickstart in first screen
- local-first security model
- screenshots/demo only if real or clearly placeholder
