# Contributing

Thanks for helping improve Pala OS.

Before changing behavior:

1. Read `AGENTS.md`, `CLAUDE.md`, `REVIEW.md`, and `.pala/rules/*`.
2. Run `npm run pala -- memory check`.
3. Make small, reviewable changes.
4. Run the relevant Pala gate and keep raw evidence.
5. Do not push, publish, delete, or write real MCP config unless a maintainer explicitly approves.

No contribution should claim PASS without command output, exit code, raw log path, changed files, risk status, and next action.
