# Pala OS Review Instructions

Review for:

- fake done claims
- missing evidence
- unrun tests
- drift between README/docs/panel/source
- unsafe config writes
- secret leakage
- hardcoded user paths
- public package claims without publication evidence
- broad refactors without characterization tests
- dashboard fake green status

Pala local gates are mandatory. Claude Code Review, if available, is optional and does not replace local gates.
