# Support

This repository is a local session pack, not a hosted service.

For help:

- Check `README.md` for quickstart commands.
- Check `docs/INSTALL.md` for local setup.
- Check `docs/MCP_CLIENTS.md` before touching MCP configs.
- Run `npm run pala -- doctor` for aggregated local read-only diagnostics.

Do not share secrets, raw logs, private local paths, or `.pala/db/pala.sqlite` in public support requests.
