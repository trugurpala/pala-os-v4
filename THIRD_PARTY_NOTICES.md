# Third-Party Notices

Pala OS v28 does not bundle third-party source code or runtime packages.

The current `package.json` uses only Node.js built-in modules. Node.js and its
built-in SQLite support are provided by the user's runtime and remain subject
to their own licenses.

Projects named in documentation, reference radar, and competitor lessons are
benchmark or research inputs only. Their code, branding, package names, copy,
and assets are not included.

GitHub Actions workflows reference GitHub-maintained actions. Review action
versions and their licenses before a public release.

Update this file whenever a dependency, copied asset, generated artifact, or
vendored component is added.
