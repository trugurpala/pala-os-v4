# Code Section Paste Guide

## Claude Code terminal or desktop Code tab

1. Open the project root.
2. Prefer strong planning mode if available:

```txt
/model opus
/effort max
```

3. Paste the contents of:

```txt
00-CLAUDE-CODE-A-YAPISTIR-TEK-METIN.md
```

or, for strong-model run:

```txt
00-CLAUDE-CODE-OPUS-4-8-MAX-CODE-BOLUMUNE-YAPISTIR.md
```

4. Do not accept vague completion.
5. Ask for evidence table, exit codes, raw logs and DB status.
