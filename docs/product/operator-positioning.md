# Operator Positioning

## Public-safe positioning

Pala OS is a local control tower for AI/vibe coding.

It does not replace coding agents. It makes Claude Code, Cursor, Codex, and other agents work with evidence, memory, token discipline, drift checks, safe MCP setup, and dashboard visibility.

## Strong internal metaphor

```txt
Pala OS helps a vibe coder operate like a disciplined senior operator:
less forgetting, less token waste, more evidence, clearer gates.
```

Avoid saying:

- best coding agent
- God mode
- unlimited autonomy
- guaranteed cost savings
- exact token cost when only estimating

Use:

- evidence-driven
- memory-aware
- cost-aware
- dashboard-visible
- local-first
- no-fake-done
