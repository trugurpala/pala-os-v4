# Recipe — Positioning + Token Discipline

## Goal

Make Pala OS public copy honest and token-economy claims evidence-backed.

## Commands

```bash
pala positioning-check
pala copy-check
pala token-economy
pala token-economy --json
```

## Expected behavior

- Pala OS is described as a control/evidence layer, not a coding agent.
- Public README uses English-first copy.
- Turkish docs may mirror the public message.
- Token usage is exact only if measured.
- Estimated usage is labelled with confidence.
- Dashboard shows known vs estimated separately.

## Evidence

```txt
command
exit code
raw log path
changed files
remaining risks
```

## Failure mode

Unsupported positioning, exact-cost language, or unmeasured savings claims block acceptance.
