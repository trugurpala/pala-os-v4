# Recipe — UI/UX Skill Radar

Goal:
Evaluate external UI/UX skill repos.

Commands:
```bash
pala skills-check
pala external-skills-refresh
pala external-skill-propose --target "ui-ux capability gap"
```

Rules:
- license/version/install method recorded
- no copying
- no auto install

Expected evidence:
- bounded local skill readiness and blockers
- local proposal only; no marketplace search
- explicit confirmation that no external skill was installed
