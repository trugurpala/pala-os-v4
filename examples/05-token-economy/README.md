# Token Economy Demo

## Goal

Record an honest token budget without inventing exact usage or cost.

## Commands

```bash
pala token-budget --goal "review local release readiness"
pala token-economy
```

## Expected evidence

SQLite and the token ledger separate known values, estimates, confidence, model, effort, and source.

## Failure / no-fake-done

Unavailable usage is Unknown, not zero. Exact cost and savings claims remain blocked without measurements.
