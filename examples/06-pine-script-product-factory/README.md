# Pine Script Product Factory Demo

## Goal

Plan a Pine Script product without unsupported financial or performance claims.

## Commands

```bash
pala plan --goal "create a Pine Script product with reproducible backtest evidence"
pala reference-check
pala docs-honesty-check
```

## Expected evidence

The decision log records risk, current references, and the requirement for reproducible backtest assumptions.

## Failure / no-fake-done

Missing backtest evidence or financial disclaimers blocks performance claims and release acceptance.
