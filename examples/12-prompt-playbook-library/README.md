# Prompt Playbook Library Demo

## Goal

Check that prompts and recipes encode the current Pala operating law.

## Commands

```bash
pala prompt-radar
pala playbook-check
pala examples-check
```

## Expected evidence

The checks report stale versions, missing commands, evidence expectations, and safety/failure paths.

## Failure / no-fake-done

Stale or incomplete playbooks remain under manual verification until corrected.
